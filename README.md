## Introduction

cpkg is a lightweight C package manager that allows you to easily install, update, and remove C packages. It is designed to be simple, easy to use, and cross-platform.

cpkg is currently in its early stages of development, and is not yet ready for general use. However, you can follow the progress on the [cpkg GitHub repository](https://github.com/chenhao2345/cpkg).


## Features and Goals

- Simple, easy-to-use command-line interface
- Cross-platform support (Windows, Linux, macOS)

## Getting Started

To get started with cpkg, you can follow the instructions below:

1. Install cpkg by following the instructions on the [cpkg GitHub repository](https://github.com/chenhao2345/cpkg).
2. Install packages by running the `cpkg -i` command.
3. Build packages by running the `cpkg -m` command.
4. Remove packages by running the `cpkg -r` command.
